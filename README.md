# _The_ 333-bot
Out of bed early to run to sick call? Testing early? Going to bother some officers at SNIT? The 333-bot will grace your Discord with alerts till you let us know! 

With handy features like weather monitoring, reminders, and (most importantly) accountability, this bot will make your your time at tech school a breeze!

# Just a heads up
This bot is built specifically for the NPS Airmen of the 333rd TRS, and the feature set reflects it's unique requirements. Feel free to grab some code snippets to help out with your own project.
